# MIPS-Assembly-Game
A 2d platformer game completely developed in MIPS assembly.<br>
Includes:<br>
  Firing a bullet by pressing B<br>
  Movement with arrow keys<br>
  Gravity and the map wraps around<br>
  Collision logic and invincibility frames<br>
